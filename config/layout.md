# Corne keymap layout

Edite as colinhas abaixo. Depois eu sincronizo o `corne.keymap` pra bater.

## Default layer

```
// | none |  Q  |  W  |  E  |  R  |  T  |   |  Y  |  U   |  I  |  O  |  P  | none |
// | none |  A  |  S  |  D  |  F  |  G  |   |  H  |  J   |  K  |  L  |  ;  | none |
// | none |  Z  |  X  |  C  |  V  |  B  |   |  N  |  M   |  ,  |  .  |  /  | none |
//                    | GUI | LWR | SPC |   | BKSP| RSE  |RSHFT|
```

## Lower layer

```
// | none |  1  |  2  |  3  |  4  |  5   |   |  6  |  7  |  8  |  9  |  0  | none |
// | none | TAB | ESC |LCTRL| LALT|  `   |   | LFT | DWN |  UP | RGT |COLON| none |
// | none | SHFT|     |     |     |      |   |     |     |  <  |  >  |QMARK| none |
//                    | GUI |     | SPC  |   | ENT |     | DEL |
```

## Raise layer

```
// | none |  !  |  @  |  #  |  $  |  %  |   |  ^  |  &  |  *  |  (  |  )  | none |
// | none | TAB |     |     | SQT |  `  |   |  -  |  =  |  [  |  ]  |  \  | none |
// | none | SHFT|     |     | DQT |  ~  |   |  _  |  +  |  {  |  }  | "|" | none |
//                    | GUI |     | SPC |   | ENT |     |LCTRL|
```
